# py-face-rekognition
Facial Recognition System using Python (>=3.6)

### Setup

```bash
git clone heisenbug237/py-face-rekognition
cd py-face-rekognition
pip install -r requirements.txt
```
### Test Image
```bash
python main.py sample.jpg
```
### Test Webcam
```bash
python main.py
```
